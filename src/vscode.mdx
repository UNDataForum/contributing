# VSCode Configuration

## Recommended settings for all OSs

| Setting | Value | Comment |
|:--------|:------|:--------|
| `editor.formatOnSave` | `true` | Turn on format on save. Required for Prettier plugin. |
| `files.autoSave` | `onFocusChange` | Save files as soon as editor looses focus. |
| `telemetry.enableCrashReporter` | `false` | Disable crash reporter. |
| `telemetry.enableTelemetry` | `false` | Disable telemetry. |

## Required settings for Windows

| Setting | Value | Comment |
|:--------|:------|:--------|
| `files.eol` | `\n` | Change line endings to LF for WSL compatibility. |
| `git.path` | `C:\\\\path\\\\to\\\\wslgit.exe` | Path to [WSLGit](https://github.com/andy-5/wslgit). |
| `terminal.integrated.shell.windows` | `C:\\\\WINDOWS\\\\System32\\\\bash.exe` | Use Ubuntu bash via WSL as terminal. |
