VSCode Configuration

Recommended settings for all OSs

SettingValueComment
editor.formatOnSavetrueTurn on format on save. Required for Prettier plugin.
files.autoSaveonFocusChangeSave files as soon as editor looses focus.
telemetry.enableCrashReporterfalseDisable crash reporter.
telemetry.enableTelemetryfalseDisable telemetry.

Required settings for Windows

SettingValueComment
files.eol Change line endings to LF for WSL compatibility.
git.pathC:\\path\\to\\wslgit.exePath to WSLGit.
terminal.integrated.shell.windowsC:\\WINDOWS\\System32\\bash.exeUse Ubuntu bash via WSL as terminal.
